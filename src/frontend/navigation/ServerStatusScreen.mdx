# ServerStatus

Shows the when the server is loading or if there is an error

<PropsTable of={ServerStatus} />

## `variant="waiting"`

Shown when returning to the app from the background and waiting for the server
to respond

<PhoneScreen>
  <ServerStatus variant="waiting" />
</PhoneScreen>

## `variant="timeout"`

Shown when we have lost communication with the server for an extended period of
time (by default 10 seconds)

<PhoneScreen>
  <ServerStatus variant="timeout" />
</PhoneScreen>

## `variant="error"`

Shown when there is an uncaught error in the server process

<PhoneScreen>
  <ServerStatus variant="error" />
</PhoneScreen>
